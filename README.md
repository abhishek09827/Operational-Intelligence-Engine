# OpsPilot.ai
Autonomous Multi-Agent DevOps Incident Copilot.

## Overview
OpsPilot is designed to autonomously detect, analyze, and resolve DevOps incidents using a multi-agent architecture.

## Architecture
- **CrewAI**: Orchestrates agents.
- **FastAPI**: Backend API.
- **Docker**: Containerization.
- **PostgreSQL**: Database.

## Setup
1. Clone the repo.
2. Run `docker-compose up --build`.

## Usage
Access the API at `http://localhost:8000`.
